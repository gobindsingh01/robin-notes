LET'S START

  • Computer Organization includes how a computer is organized or to put it simply how all the components of computers are made and how they are working.

  • Assembly Language is a computer language that is close to computers than other computer languages.
  • High level language is closer to the human
  • Low level language is closer to the machine.

The first thing introduced to the student is VON NEUMANN ARCHITECTURE. what is it? lets find out.

VON NEUMANN ARCHITECTURE (click to open)
  • VON NEUMANN ARCHITECTURE is a graphical representation (diagram) of computer components and how they connect.
  • As we can see in the above diagram, the architecture consists of 3 parts :
    • Input Device : helps in taking inputs , examples - keyboard, mouse, touch screen, microphone, etc.
    • Processing Unit : also called as CPU (central processing unit) helps in processing the input data and give the output back to user.This is also called as brain of computer as this is where all the working happens and everything is connected to this brain.
    • Output Device : helps in giving output back to user , examples - monitor, speakers, projectors, etc.

    • Now that we know the basics, lets move a little step deeper. let's see control unit ,

  • Control Unit : helps controlling the flow of the data, think of it as a component which takes data from input and work on the data appropriately and then send it back to the output device.
  • Arithmetic and Logic unit :
    Arithmetic unit helps in doing mathematical work on the input and produce the output. example -> +,-,/,etc.
    Logic unit helps in working on logical work. example - >, <, >=, <=,etc.
